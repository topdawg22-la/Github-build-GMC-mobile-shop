# GMC Mobile Service Rig - Front Lighting Utility Upgrade

## Overview
Custom installation of a high-output LED light bar onto a GMC brush guard. This build is designed for high-vibration equipment service environments, prioritizing structural rigidity and permanent hardware retention.

## Specifications
* **Base Material:** 34.5" Steel Angle Iron (Finished in Safety Red)
* **Mounting Span:** 27.5" (Bolt-hole to Bolt-hole)
* **Alignment:** 3.5" symmetrical overhang on each side
* **Hardware:** New Grade-specific bolts with Lock Washers
* **Thread Retention:** Permanent Threadlocker (Red) for high-vibration resistance

## Installation Notes
1.  **Preparation:** Used bed-mounted vise for precision drilling of pilot and main holes.
2.  **Corrosion Control:** Pre-painted angle iron; holes touched up post-drilling.
3.  **Torque:** Applied Red Threadlocker to clean threads; lock washers crushed flat for maximum tension.
